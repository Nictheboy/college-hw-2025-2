## 第二题

### 第一问

200ps

### 第二问

L(I-Mem) + L(Sign-Extend) + L(Shift-Left-2) + L(Add) + L(Mux)

= 200 + 15 + 10 + 70 + 20

= 315 (ps)

### 第三问

L(I-Mem) + Max(L(Sign-Extend) + L(Shift-Left-2) + L(Add), L(Regs) + L(Mux) + L(ALU)) + L(Mux)

= 200 + Max(15 + 10 + 70, 90 + 20 + 90) + 20

= 200 + Max(95, 200) + 20

= 200 + 200 + 20

= 420 (ps)

### 第四问

跳转指令。

### 第五问

无条件跳转指令。

### 第六问

L(I-Mem) + Max(L(Sign-Extend) + L(Shift-Left-2) + L(Add), L(Regs) + L(Mux) + L(ALU)) + L(Mux)

= 200 + Max(15 + L(Shift-Left-2) + 70, 90 + 20 + 90) + 20

= 220 + Max(85 + L(Shift-Left-2), 200)

= 420 + Max(L(Shift-Left-2) - 115, 0)

若 L(Shift-Left-2) 增加到 115ps 以上，周期会增加 L(Shift-Left-2) - 115ps，反之则周期不变。
